# Rex Code AI V1

**The field tool that stops red tags before inspection.**

No code book. No 400 pages of IMC. Just: `Will this pass?` → `Fix Now` → `Copy for Report`.

### **What It Is**
Rex Code AI is a mobile-first code checker for HVAC techs. Built for job sites in work boots. 
Base: **2024 IMC / 2024 IECC** with 2021 blend coverage so it works anywhere in the state.

### **What’s Inside V1**
| Feature | Why Techs Use It |
| --- | --- |
| **FAIL / CHECK / PASS Cards** | Thumb knows before brain does. Red = stop. Yellow = measure. Green = go. |
| **Field Fix First** | 1 line action. Cut duct. Add MUA. Change tape. No theory. |
| **Inspector Rebuttal** | Crew only. Script to cite code on site without fighting. |
| **📋 Copy for Report** | Crew only. 1-tap line for invoices, notes, or pushback. |
| **Search by Problem** | Type `dryer vent` or `clearance`. Get the card in 2 seconds. |

### **Coverage**
**100 highest red-tag-risk codes** across:
1.  **Exhaust + Make-up Air** - Hoods, Dryers, Bath fans
2.  **Combustion Air + Venting** - Furnaces, Water heaters 
3.  **Condensate + Drains** - Attic units, Secondary pans
4.  **Duct Sealing + Insulation** - IECC C403/C406 energy fails
5.  **Access + Clearance** - 30" rule, label clearances

**Note**: Base code is 2024 IMC/IECC. Always verify local AHJ amendments.

### **Tiers**
1.  **SOLO** - See all cards. 1 device. 
2.  **CREW** - Everything in Solo + Copy button + Rebuttal scripts. 5 devices.

### **How to Use**
1.  Open the Vercel link
2.  Enter your `SOLO-XXXX` or `CREW-XXXX` key
3.  Search: `dryer vent`, `tape`, `clearance`, `MUA`
4.  Fix it. Pass inspection.

### **Disclaimer**
This tool is a field reference based on 2024 IMC/IECC. It is not legal advice. AHJs may have local amendments. When disputed, ask the inspector to cite the local amendment number on site.
